 Application Patterns

There is a growing concern in projects to improve and standardise the development process, create synergies between projects and use tools and processes to improve the quality of the code being developed. One of the recurring questions is which tools are the most adequate to use in the different steps of the projects and how can the development process be enhanced with tasks such as Unit Tests, Continuous Integration, Code Checking, etc.
